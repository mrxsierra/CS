# Cybersecurity Engineer Interview Track (2026-27)

## 1. Role Overview
Focuses on preventing system intrusions, securing API breaches, and managing identity/access in a world of high ransomware risk.

## 2. 2026-27 Ecosystem Focus
- **Network Security**: TCP/IP, DNS, SSH, Firewalls.
- **Application Security**: OWASP Top 10 (SQLi, XSS, SSRF), API security auditing.
- **Security Operations (SecOps)**: SIEM tools (Splunk), IAM permissions (Least Privilege).

## 3. 12-Week Learning Pathway
- **Week 1-3: Networking & Linux Hardening**: Mastering routing, SSH configs, and directory permissions.
- **Week 4-7: App Security & Threat Hunting**: Identifying vulnerabilities and auditing source code.
- **Week 8-12: SecOps & IAM**: Traffic log analysis, intrusion detection, and IAM configuration.
