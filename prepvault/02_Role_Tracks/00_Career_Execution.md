# Career Execution Strategy: The 90-Day Engine

In the 2026-27 market, applying to 500+ jobs via generic portals is a failing strategy. You must treat your job hunt as a disciplined engineering project.

## 1. The 90-Day Compliance Schedule

```
               THE 90-DAY COMPLIANCE SCHEDULE
  ┌─────────────────────────────────────────────────────────┐
  │ DAYS 1-30: Foundation & Core Stack                      │
  │ - Master 1 Language Internals (JS/Java/Python/Go)       │
  │ - Learn Core Database Systems (ACID, Indexing)          │
  │ - Deploy Project 1 (API with DB & Documentation)        │
  ├─────────────────────────────────────────────────────────┤
  │ DAYS 31-60: High-Level Engineering                      │
  │ - Add Tests, CI/CD, and Setup Logging to Projects       │
  │ - Practice 2 DSA Patterns Daily (Focus on Adaptation)   │
  │ - Clean up LinkedIn & GitHub (Proof of Work)            │
  ├─────────────────────────────────────────────────────────┤
  │ DAYS 61-90: Active Application Phase                    │
  │ - Tailor Resume for every Target JD (ATS Optimization)  │
  │ - Execute 5 Targeted Cold Outreaches Daily (LinkedIn)   │
  │ - Run 2 Mock Interviews Weekly (Thinking Aloud)         │
  └─────────────────────────────────────────────────────────┘
```

## 2. Professional Outreach & Networking
Stop sending "Refer me" messages to strangers. Build professional relationships instead.
- **Warm Referrals**: Ask for advice or technical insight first. 
- **Cold Outreach Strategy**:
    - "I saw your team is building X. I recently built a project using similar tech (link) and would love to hear your thoughts on how you handle Y bottleneck."
- **LinkedIn Optimization**: Your profile should act as a portfolio. Post technical snippets, project milestones, and "What I learned today" insights.

## 3. Resume Tailoring (ATS Optimization)
- **Proof of Work**: Include links to live deployments, GitHub repos, and documentation.
- **Keywords**: Ensure your resume contains the specific technologies mentioned in the JD (e.g., "Fastify", "Kubernetes", "gRPC").
- **Impact Metrics**: Use the formula: *Accomplished X as measured by Y, by doing Z.*
    - *Example*: "Reduced API latency by $40\%$ as measured by New Relic by implementing Redis caching on high-traffic routes."

## 4. Managing the Emotional Landscape
The job search is mentally exhausting. Success requires a survival system:
- **Habit-Driven Routine**: 4 hours deep work (DSA/Projects), 2 hours operations (Networking/Tracking).
- **Interview Feedback Log**: Every rejection is a diagnostic data point. Use the [[05_Templates/interview_feedback_template|Interview Feedback Template]] to document every round.

## 5. The Top of the Funnel
If you are not getting calls, your pipeline is failing at the top.
1. **Resume Filter**: Are you passing the automated ATS? (See Section 3).
2. **Pedigree Gate**: Are you relying too much on portals? Start securing referrals.
3. **Skill Visibility**: Is your GitHub active with production-grade code?
