# 💻 Coding Rounds

## Description
What to expect in technical coding interviews — the most common round across all tech companies. You'll be asked to solve algorithmic problems (often on a whiteboard or shared editor) while explaining your thought process.

## Key Concepts
- **Whiteboarding**: Writing code by hand or on a shared virtual whiteboard. Focus on structure and clarity.
- **Pair Programming**: Collaborative coding with the interviewer. They may give hints or ask you to change approach.
- **LeetCode-Style Problems**: Medium to Hard difficulty. Common patterns include Trees, Graphs, DP, and Sliding Window.
- **Time & Space Analysis**: You must analyze and explain your Big-O complexity.
