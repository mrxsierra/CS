# 🎨 Frontend Stack Index

## Overview
Master the modern frontend ecosystem — from markup and styling to frameworks and build tools. Each deep dive is interview-focused: expect "how it works" over "how to use it."

## How to Use These Deep Dives
1. **Review prerequisites** — Make sure you're comfortable with [[02_role_tracks/02_frontend_engineer|Frontend Engineer Track]] foundations
2. **Read in order** — Start with HTML/CSS, then JS/TS, then React, then Next.js, then Build Tools
3. **Practice implementing** — Each module has common interview questions — try building them from scratch
4. **Connect to foundations** — These stack modules build on [[01_foundations/05_networking|Networking]], [[01_foundations/04_operating_systems|OS]], and [[01_foundations/01_dsa|DSA]]
