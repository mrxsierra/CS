# AI Orchestration (RAG, LangChain, Vector Databases)

Modern engineering requires building *with* AI. This module covers the architectural patterns for AI-native applications.

## 1. Retrieval-Augmented Generation (RAG)
- **Concept**: Connecting LLMs to private/external data.
- **Components**: Document Chunking, Embedding Models, Vector Search, Prompt Augmentation.

## 2. Agentic AI Workflows
- **LangChain / LlamaIndex**: Frameworks for building agents with tool-calling capabilities.
- **Stateful Chains**: Managing context across multiple AI interactions.

## 3. Vector Databases
- **Core Stack**: Pinecone, Qdrant, Milvus, or `pgvector`.
- **Indexing Strategies**: Semantic search vs. Keyword search.

## 4. Evaluation & Observability
- Monitoring for performance drift and hallucinations.
- Benchmarking LLM responses.
