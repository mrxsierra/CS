# Math & Geometry

## Overview
Math and Geometry problems focus on numerical properties, bitwise operations, and spatial manipulation of matrices/coordinates.

## Key Principles & Patterns
- Modulo Arithmetic: Handling large numbers to avoid overflow.
- Matrix Manipulation: In-place rotation and traversal (Spiral, Zigzag).
- Prime Numbers: Sieve of Eratosthenes, Prime factorization.
- Coordinate Geometry: Slopes, distances, and area.

## Complexity Analysis
Varies (e.g., O(n) for matrix traversal, O(log n) for power/GCD).

## Common Interview Questions
1. **Rotate Image**:  Rotate a matrix by 90 degrees in-place.
2. **Spiral Matrix**:  Traverse a matrix in a spiral order.
3. **Happy Number**:  Use a Hash Set or Fast/Slow pointers to detect cycles in number sequences.
4. **Plus One**:  Simple array manipulation with carry logic.
5. **Pow(x, n)**:  Use Binary Exponentiation for O(log n).

## Role-Specific Applications
- **Frontend**: Canvas/SVG manipulations, CSS transforms, animations and easing functions.
- **Backend**: Cryptography, hashing algorithms, financial calculations (precision matters!), spatial databases (PostGIS).
- **ML/Data**: Linear algebra (matrices), probability and statistics, optimization algorithms.
- **DevOps**: Resource calculation, cost modeling, rate limiting algorithms (Leaky/Token Bucket).
