# 🗄️ Databases Stack Index

## Overview
Database knowledge is essential for every backend-adjacent role. These deep dives cover the three pillars of modern data storage: relational (PostgreSQL), document (MongoDB), and caching (Redis).

## Stack Modules

| # | Module | Focus Areas | Interview Weight |
|---|--------|-------------|-----------------|
| 01 | [[01_SQL_PostgreSQL.md\|PostgreSQL & SQL]] | ACID, Window Functions, Indexing, Query Plans, CTEs, MVCC | ⭐⭐⭐⭐⭐ |
| 02 | [[02_MongoDB_NoSQL.md\|MongoDB & NoSQL]] | Document Model, Aggregation Pipeline, Sharding, Embedding vs. Referencing | ⭐⭐⭐⭐ |
| 03 | [[03_Redis_Caching.md\|Redis & Caching]] | Data Structures, Cache Patterns, Eviction, Persistence, Distributed Locks | ⭐⭐⭐⭐ |

## How to Use These Deep Dives
1. **Start with SQL** — it's the most common database interview topic
2. **Understand the tradeoffs** — know when to pick SQL vs. NoSQL vs. Cache
3. **Practice the patterns** — implement cache-aside, rate limiting, and aggregation pipelines
4. **Connect to foundations** — these build on [[01_Foundations/09_SQL_Database_Deep_Dive|SQL Database Deep Dive]] and [[01_Foundations/03_System_Design|System Design]]
