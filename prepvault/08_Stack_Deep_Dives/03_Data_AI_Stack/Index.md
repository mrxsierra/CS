# 🤖 Data & AI Stack Index

## Overview
The Data & AI stack covers the full spectrum from data analytics to deep learning, LLMs, and MLOps. These deep dives prepare you for interviews across Data Science, ML Engineering, and AI Engineering roles.

## How to Use These Deep Dives
1. **Start with the foundations** — Python for Data Analytics → Data Science → ML
2. **Deep dive into specialization** — Deep Learning → LLMs (for AI/ML roles) or Spark (for Data Engineering)
3. **Finish with MLOps** — Model deployment and monitoring
4. **Connect to foundations** — These build on [[01_Foundations/01_DSA|DSA]], [[01_Foundations/03_System_Design|System Design]], and [[02_Role_Tracks/04_ML_Engineer|ML Engineer Track]]
